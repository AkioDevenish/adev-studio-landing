# Case Study Images

This folder contains all images for case study presentations.

## Folder Structure

### Met Office Trinidad & Tobago (`/met-office/`)
- `homepage-desktop.png` - Desktop homepage view (shots.so mockup)
- `weather-data.png` - Weather data section detail (shots.so mockup)
- `mobile-view.png` - Mobile responsive view (shots.so mockup)
- `satellite-radar.png` - Satellite and radar interface (shots.so mockup)
- `marine-conditions.png` - Marine weather section (shots.so mockup)
- `hero-image.jpg` - Main hero image for case study

### USC Connect (`/usc-connect/`)
- `app-overview.png` - Mobile app overview (shots.so mockup)
- `features.png` - Key features showcase
- `hero-image.jpg` - Main hero image for case study

## Image Guidelines

### shots.so Mockups
- **Format**: PNG with transparent backgrounds preferred
- **Resolution**: High-resolution (2x) for retina displays
- **Style**: Professional mockup frames from shots.so
- **Naming**: Descriptive, lowercase, hyphen-separated

### Hero Images
- **Format**: JPG for photographs, PNG for graphics
- **Resolution**: Minimum 1920x1080 for hero sections
- **Aspect Ratio**: 16:9 for desktop, 4:3 for project cards
- **Optimization**: Compressed for web performance

## Usage in Components

Images are referenced from the case study components using:
```jsx
src="/images/case-studies/met-office/homepage-desktop.png"
```

## shots.so Workflow

1. Visit https://shots.so/
2. Enter the project URL
3. Select appropriate device/browser frame
4. Customize styling and effects
5. Export as high-resolution PNG
6. Save to appropriate case study folder
7. Update component image paths
